Controls:

Mouse controls:

 - Left-click/Tap: Drag workspace, Drag person, Select person

 - Right-click/Hold: Drag workspace, Open context menu

 - Middle-click: Drag workspace

 - Scroll-wheel/Pinch: Zoom in or out

 - Double-click/Double-tap: Add person, Change person's name/gender

Keyboard controls:

 - N, P, S, C, A: Adds new person, parent, spouce, child, sibling respectively

 - Backspace, Delete: Deletes selected person

 - E: Opens edit menu for selected person

 - G: Opens gender menu for selected person

 - Alt + M, N, F: Selects male, non-binary, female gender respectively for selected person while gender menu is opened

 - Escape: Hide menu, clear selection

 - Arrow keys/Tab: Selects person

 - Ctrl + Z, Y: Undo, Redo

 Use Alt + (key) to control while name is being edited

Actions:

 - Add person: Adds a person connected to the last person added or creates the first person

 - Add parent: Either creates a new parent group or adds a parent to an existing group

 - Add spouce: Adds a spouce who shares the person's children

 - Add child: Adds a child who is siblings with the rest of the children

 - Add sibling: Adds a sibling to a family, prioritising those with parents & children

 - Edit: Opens menu to edit the person's relationships

 - Reset all positions: Reset the positions of all people and connection points

 - Reset position: Resets the position of chosen element (person or connection point)

 - Undo & Redo: Has a limit of 20
